## Descrição

Esta estilização combina a paleta de cores vintage de "Apothecary"
com a estrutura tipográfica e de layout de "Garments".
O layout utiliza **CSS Grid** para a organização principal e **Flexbox**
para o alinhamento de componentes.
A responsividade advém de **Media Queries**, adaptando o layout de
duas colunas para uma coluna única em dispositivos menores.

## Recursos Utilizados

- **CSS Grid Layout** para a estrutura da página.
- **Flexbox** para alinhamento de componentes.
- **Variáveis CSS** para gerenciamento de cores, fontes e espaçamento.
- **Pseudo-elementos** para elementos decorativos e
  extensões de texto sem alterar o HTML.
- **Media Queries** para design responsivo.
- **Josh Comeau Reset** para uma base consistente entre navegadores.
- **Google Fonts** para tipografia personalizada.
- **Modo Claro/Escuro automático** através da media query `prefers-color-scheme`.
