Seção 2 - Ranking de tecnologias (lista ordenada reversa)

  1. SQL

    Linguagem padrão para gerenciar e manipular bancos de dados relacionais, fundamental para o desenvolvimento de aplicações web dinâmicas que processam dados.

  2. Git

    Ferramenta essencial para o controle de versão e colaboração em projetos de software, permitindo rastrear mudanças no código-fonte ao longo do tempo.

  3. JavaScript

    Linguagem de programação que permite a criação de conteúdo interativo e dinâmico no lado do cliente, sendo um dos pilares do desenvolvimento web moderno.

  4. CSS

    Responsável pela estilização e layout das páginas web, permitindo separar o conteúdo da apresentação visual e garantindo a responsividade do site.

  5. HTML

    A base fundamental da web. Define a estrutura e o significado do conteúdo através de elementos semânticos, sendo o ponto de partida obrigatório para qualquer desenvolvedor.

Seção 3 - Glossário de termos técnicos (lista de definição)

HTML
HyperText Markup Language (Linguagem de Marcação de Hipertexto).
Linguagem padrão utilizada na criação de páginas e documentos para a World Wide Web.
CSS
Cascading Style Sheets (Folhas de Estilo em Cascata), utilizada para definir a apresentação visual de documentos HTML.
Navegador
Browser
Software projetado para permitir a navegação e visualização de páginas da web, processando códigos HTML, CSS e JavaScript.
API
Application Programming Interface (Interface de Programação de Aplicações).
Conjunto de normas que possibilitam a comunicação entre plataformas por meio de uma série de padrões e rotinas.
Tag
Etiqueta
Elementos de marcação usados para instruir o navegador sobre como a estrutura do conteúdo deve ser exibida.
JavaScript
Linguagem de programação interpretada que permite implementar itens complexos e interatividade em páginas web.
HTTP
Hypertext Transfer Protocol, o protocolo base utilizado para a troca de informações na internet.
URL
Uniform Resource Locator, o endereço único que identifica um recurso na rede.
